# TP2 : Architecture MVC

## Depuis le TP1, devoir migrer le projet vers l'architecture MVC.

## Développement:

### • Créer les routes du projet avec des constructeurs et des méthodes d'appel.

### • S'assurer d'avoir des contrôleurs par fonctionnalité.

### • Créer des modèles pour toutes vos tables.

### • Développer au moins 4 vues : une vue statique comme page d'accueil et d'autres vues dynamiques pour activer les fonctions CRUD.

### • Utiliser TWIG pour rendre vos vues.

### • Pouvoir utiliser une seule table pour appliquer les fonctions CRUD, mais le choix de la table doit refléter la complexité du projet.

### • Valider les données.

### • S'assurer que le projet est fonctionnel, avec la navigation, la gestion des erreurs et la conception CSS intégrée.
